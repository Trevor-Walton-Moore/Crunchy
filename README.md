# Crunchy

Crunchy is a website clone of Chewy.com.

Live site: [Crunchy](https://crunchy-63ov.onrender.com)

![crunchy-example-pic](https://user-images.githubusercontent.com/106632610/217108922-c0a0087e-52ba-479d-a71f-e5c01b63329e.png)

* JavaScript
* React
* Redux
* Python
* Flask
* HMTL
* CSS
* SQLAlchemy
* SQL
* Node.js
* Postgress
* Render
